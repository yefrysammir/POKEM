# Monster GO 🎮

Un juego web progresivo (PWA) inspirado en Pokémon GO, desarrollado con HTML, CSS y JavaScript vanilla.

## 🚀 Características

- **Mapa Virtual 2D**: Explora un mundo generado proceduralmente
- **Sistema de Captura**: Mecánicas de lanzamiento con diferentes Poké Balls
- **Combate en Tiempo Real**: Batallas por turnos con sistema de tipos elemental
- **Raids**: Incursiones contra Pokémon legendarios de 1-5 estrellas
- **Evoluciones**: Sistema completo de evolución con caramelos e items
- **Inventario**: Mochila con gestión de items, medicinas y mejoras
- **Tienda**: Compra de items y monedas premium
- **PWA**: Funciona offline con Service Worker
